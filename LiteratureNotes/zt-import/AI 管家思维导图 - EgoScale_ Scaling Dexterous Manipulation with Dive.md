# EgoScale: Scaling Dexterous Manipulation with Diverse Egocentric Human Data

## 研究背景与目标
- 人类行为是学习物理智能的最可扩展数据源之一
  - 但之前的人到机器人转移局限在小规模数据（数十到数百小时）
  - 且多聚焦于夹爪或低自由度手，缺乏精细手指运动
- 核心目标
  - 证明大规模人类数据能有效支持高自由度灵巧操作
  - 提出EgoScale框架，利用20,854小时自我中心视频训练VLA模型
  - 发现人类数据规模与性能之间的缩放定律
  - 实现高效的跨实体迁移和一次性任务适应

## 研究方法
- 人类数据收集与处理
  - 第一阶段：大规模自我中心人类预训练数据（20,854小时，覆盖9,869场景、6,015任务、43,237物体）
  - 第二阶段：对齐的人-机器人中训练数据（约50小时人类 + 4小时机器人，344个桌面任务）
  - 动作标注：通过SLAM和手部姿态估计提取手腕运动和手部关节
- 人类动作表示
  - 手腕层面：相对手腕运动（SE(3)变换），消除全局摄像机运动影响
  - 手部手指：通过优化重定向将21个人手关键点映射为22-DoF灵巧手关节空间
- 模型架构
  - 基于流的视觉-语言-动作（VLA）模型（类似GR00T N1）
  - 视觉语言骨干 + 扩散变换器（DiT）动作专家
  - 轻量级实体条件MLP适配器处理不同机器人的状态和动作空间
- 训练流程（三个阶段）
  - 第一阶段：人类预训练（20K小时，100K步，256 GB200 GPU）
  - 第二阶段：对齐中训练（50K步，冻结语言骨干，更新视觉编码器和DiT）
  - 第三阶段：后训练（任务特定机器人演示，10K步）
- 机器人系统
  - 主要平台：Galaxea R1Pro人形机器人 + 22-DoF Sharpa灵巧手
  - 跨实体评估：Unitree G1机器人 + 7-DoF三指手

## 关键研究结果
- 大规模人类预训练是强灵巧操作策略的关键
  - 人类预训练 + 中训练在所有任务上平均成功率提升54%（相对无预训练基线）
  - 即使仅有大规模人类预训练（无中训练），也优于仅中训练基线（图4）
- 策略性能随预训练数据量扩大而提升
  - 验证损失与数据量呈对数线性关系：L = 0.024 − 0.003·ln(D)，R²=0.9983（图5）
  - 下游机器人任务完成分数从1k小时的0.30提升至20k小时的0.71，无饱和迹象
- 对齐的人-机器人中训练实现一次性转移
  - 仅用单一机器人演示，折叠衬衫成功率88%，拧瓶盖成功率55%（图6）
  - 省略人类预训练或中训练均失败
- 人类预训练支持跨实体迁移
  - 在Unitree G1机器人上，预训练+中训练显著优于仅用G1数据训练（图7）
  - 学习到可复用的、与实体无关的运动先验
- 手部动作表示设计影响性能
  - 全关节重定向（22-DoF）优于仅手腕或指尖表示（图8）
  - 指尖表示因映射误差导致不稳定的抓取

## 研究结论与意义
- 核心贡献
  - 首次在超20K小时人类数据上发现灵巧操作的缩放定律
  - 提出简单有效的人到机器人转移方案：大规模预训练 + 少量对齐中训练
  - 实现一次性迁移到新灵巧任务和跨实体泛化
- 广泛影响
  - 人类数据可作为机器人学习的可预测、可扩展监督源
  - 人类可被视为另一个可扩展的实体，大幅提升机器人数据效率
- 局限性
  - 未探索超越20K小时和当前模型容量的极限
  - 未来方向：结合弱监督视频、缩小实体差距、实现零样本执行
